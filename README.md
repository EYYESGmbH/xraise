# xraise
**Sicherheitsargumentation mittels  erklärbarer KI für das automatisierte Fahren im Bahnbetrieb**

Dieses Repository enthält den Source Code zum Projext XRAISE

Projektziel: 
------------

Ziel des Projekts ist es, möglichst konkrete Ansätze zu identifizieren und zu bewerten, ob und wie Methoden der XAI den Nachweis für ein Hinderniserkennungssystem bei GoA3+ unterstützen können. Die Projektergebnisse sollen dazu beitragen, ein praxistaugliches Verfahren für den künftigen Nachweis bei GoA3+ Methodik (Xraise) zu entwickeln und zu normieren, wie es für den Nachweis der Sicherheit bei „klassischer“ Software in der Norm DIN EN 50657 (DIN EN 50657 VDE 0831-657, 2017) definiert ist. 

zum Forschungsbericht: <https://www.dzsf.bund.de/DZSF/DE/Forschungsergebnisse/Forschungsberichte/forschungsberichte_node.html>
